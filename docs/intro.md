# OpsTrack 指南概览

OpsTrack 是一份运维工程师的工作记录与知识沉淀手册，围绕 Kubernetes、Prometheus、DevOps、Shell/Python 工具与数据库维护等核心议题构建。我们以「巡检记录 + Playbook + 自动化流水线」为主线，为日常集群运营、告警响应与持续交付提供统一视图。

## 你能在这里找到什么？

- **平台巡检**：Kubernetes 控制平面、节点、网络与存储的健康检查及常见异常处理流程。
- **可观测矩阵**：Prometheus、Loki、Tempo、Grafana 等指标/日志/追踪的整合方案与告警模板。
- **DevOps 与脚本工具**：CI/CD 流水线、GitOps 标准与 Shell/Python 工具库，帮助自动化重复操作。
- **数据库运维**：MySQL、PostgreSQL、Redis 等数据库的备份、恢复、性能调优与故障排查指南。
