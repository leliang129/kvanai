# 工作日志（工具 & 故障）

这里不写“日记体”，只记录可复用的运维工具链路与可复盘的故障处置过程。

## 目录

- [运维工具索引](/docs/journal/opstools)
- [故障记录索引](/docs/journal/faultrecord)

## 记录规范

- **时间线**：发生时间、发现方式、影响范围、恢复时间
- **信号**：关键指标/日志/追踪，截图用链接或关键字段替代
- **处置**：执行过的命令、变更点、回滚策略
- **复盘**：根因、改进项、后续跟踪（Owner / 截止时间）
