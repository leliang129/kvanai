# 08 网络

## 网络基础（TCP/IP）

基础概念：

- IP：主机地址
- TCP/UDP：传输层协议
- 端口：进程通信端点
- 路由：数据包转发路径

常用查看命令：

```bash
ip addr
ip route
ping -c 4 8.8.8.8
```

## `curl` / `wget`

```bash
curl -I https://example.com
curl -sS https://example.com/health
curl -o file.tar.gz https://example.com/file.tar.gz
wget https://example.com/file.tar.gz
```

调试 HTTP：

```bash
curl -v https://example.com
curl -w "\ncode=%{http_code} time=%{time_total}\n" -o /dev/null -s https://example.com
```

## 端口与连接（`netstat/ss`）

`ss` 是新系统推荐工具：

```bash
ss -lntp        # 监听端口
ss -antp        # 全部 TCP 连接
ss -s           # socket 统计
```

旧工具：

```bash
netstat -lntp
```

排查端口占用：

```bash
sudo lsof -i :8080
```

## DNS

```bash
dig example.com +short
nslookup example.com
cat /etc/resolv.conf
```

常见问题：

- DNS 解析慢导致应用超时。
- 本地缓存与权威记录不一致。

## 防火墙（`iptables/ufw`）

Ubuntu 常用 `ufw`：

```bash
sudo ufw status
sudo ufw allow 22/tcp
sudo ufw allow 80,443/tcp
sudo ufw enable
```

`iptables` 基础查看：

```bash
sudo iptables -L -n -v
```

安全原则：

- 默认拒绝入站，按需开放端口。
- 规则变更前先确认回滚路径，避免把自己锁在机器外。
