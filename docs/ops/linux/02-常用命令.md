# 02 常用命令

## 文件操作（`ls/cp/mv/rm`）

```bash
ls -alh                # 列出详细信息
cp -av src/ dst/       # 保留属性并显示过程
mv old.txt new.txt     # 重命名
rm -rf /tmp/demo       # 递归删除（高危）
```

建议：

- 危险删除前先 `ls` 校验路径。
- 生产环境避免裸 `rm -rf`，建议先移动到回收目录。

## 查找（`find/locate`）

`find` 实时扫描，精确但可能慢：

```bash
find /var/log -type f -name "*.log" -mtime -1
find . -type f -size +100M
```

`locate` 基于索引，快但可能不实时：

```bash
locate nginx.conf
sudo updatedb
```

## 查看（`cat/less/head/tail`）

```bash
cat /etc/os-release
less /var/log/syslog
head -n 20 app.log
tail -n 100 app.log
tail -f app.log
```

排障常用组合：

```bash
tail -f app.log | grep -E "ERROR|WARN"
```

## 压缩（`tar/gzip`）

```bash
tar -czf logs-$(date +%F).tar.gz /var/log/myapp
tar -xzf logs-2026-03-25.tar.gz -C /tmp/restore
```

参数速记：

- `c` 创建，`x` 解压，`z` gzip，`f` 指定文件。

## 磁盘（`df/du`）

```bash
df -h                # 分区容量
du -sh /var/log/*    # 目录体积统计
```

快速定位大文件：

```bash
find /var -type f -size +500M -exec ls -lh {} \;
```

常见坑：

- 日志已删除但进程仍持有句柄，空间不释放。
- 用 `lsof | grep deleted` 定位后重启对应进程。
